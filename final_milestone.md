# Milestone: Tutorial and Community Review + Final Publication, Outreach & Sustainability

## Milestone Outputs

- **Final close-out report**
- **Final closeout video**
- All lessons, video tutorials, and guidelines will be edited into an official open-source learning hub.
- GitHub repo will include a homepage docs (MkDocs or Docusaurus) for easy navigation.
- A public online workshop will introduce the results and guide new developers on getting started.
- Establishes a sustainability plan: plan detailing how community contributors can continue adding new lessons and updates, set up GitHub sponsorship or donation links for community support.
- Video tutorial is recorded with voiceover in Vietnamese, with English subtitles.
- **Community review process:** opening GitHub issue templates so anyone can provide feedback or request clarification. A Discord channel will allow learners to discuss directly with the team.

## Acceptance Criteria

- Final closeout report is publicly available.
- Final closeout video is publicly available.
- Learning hub website running (direct link to repo docs).
- Repo contains all core + advanced lessons, video references, and security guidelines.
- Public workshop with at least 50 attendees.
- Workshop recording uploaded publicly.
- Maintainer list + sustainability plan visible in repo.
- Each video must have Vietnamese voiceover + English subtitles.
- Community review channel active, with at least 30 members and 10 valid feedback submissions.
- Issue template merged into main repo, demonstrating open review.
