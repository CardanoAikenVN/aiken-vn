# UI Components Documentation

This directory contains documentation for the Aiken-VN landing page UI components, design patterns, and implementation guidelines.

## Component Categories

### Timeline Components
- **QuestTimeline** - Main timeline container for sequential quest display
- **QuestCard** - Individual quest card with content and illustration

### Design Patterns
- **Speedrun Pattern** - Asymmetrical 40/60 layout with centered timeline
- **Pixel Design** - Retro pixel-art themed components

## Quick Links

- [Quest Timeline Design Pattern](./quest-timeline-design.md)
- [Component API Reference](./component-api.md)
- [Responsive Design Guidelines](./responsive-guidelines.md)

## Getting Started

All components are located in `/src/components/LandingPage/` and follow TypeScript with React 19 best practices.

## Contributing

When adding new UI components:
1. Document the component's purpose and use cases
2. Include responsive behavior specifications
3. Provide accessibility (ARIA) labels
4. Include code examples
5. Update this index with links to new documentation
