# Phần 4: Mint Token & NFT (Minting Tokens & NFTs)

Tự tạo đồng tiền hoặc tác phẩm số độc đáo! Bạn sẽ viết *Minting Policy* để quản vòng đời Native Asset. Dù là token có thể thay thế hay NFT độc nhất, bạn kiểm soát cách tạo và đốt trên mạng để chia sẻ với bạn bè.
