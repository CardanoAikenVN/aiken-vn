# Phần 2: Kiến Trúc Cardano (Cardano Architecture)

Siêu sức mạnh của Cardano nằm ở kiến trúc *eUTxO* độc đáo. Bạn sẽ đi sâu vào mô hình lưu trữ để hiểu sự khác biệt với chuỗi account-based. Sau đó, bạn khám phá cơ chế truyền dữ liệu giữa giao dịch bằng *Datum* và *Redeemer*.
