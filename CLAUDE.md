## Project Overview

Vietnamese Aiken is a Docusaurus-based documentation site teaching Aiken smart contract development for Cardano in Vietnamese. It features a custom retro-themed landing page with React components.

## Development Commands

```bash
npm install          # Install dependencies
npm start            # Development server at localhost:3000
npm run build        # Production build to build/
npm run serve        # Preview production build locally
npm run clear        # Clear Docusaurus cache (useful for troubleshooting)
```

## Architecture

### Entry Points
- [src/pages/index.js](src/pages/index.js) - Homepage that renders the custom LandingPage component
- [docusaurus.config.js](docusaurus.config.js) - Site configuration (Vietnamese locale, Mermaid diagrams, TailwindCSS plugin)
- [sidebars.js](sidebars.js) - Autogenerated sidebar from docs directory

### Landing Page System
The landing page uses a quest/speedrun visual metaphor with these key components in [src/components/LandingPage/](src/components/LandingPage/):

- `index.tsx` - Main orchestrator composing Hero, QuestTimeline, OnboardingTrain, Footer, Starfield
- `QuestTimeline.tsx` / `QuestCard.tsx` - Timeline visualization of learning phases
- `constants.ts` - Quest data definitions (QUESTS_BATCH_1, QUESTS_BATCH_2)
- `translations.ts` - Vietnamese i18n strings
- `Hero.tsx`, `Footer.tsx` - Page sections
- `Starfield.tsx`, `OnboardingTrain.tsx`, `PixelComponents.tsx` - Visual effects

### Documentation Structure
Documentation lives in [docs/](docs/) with numbered directories for ordering:
- `00-ui-components/` - UI component documentation and design patterns
- `01-moi-truong/` - Environment setup
- `02-cu-phap/` - Aiken syntax
- `03-tu-duy-eutxo/` - eUTxO thinking
- `04-smart-contract-dau-tien/` - First smart contract
- `05-testing/` - Testing
- `06-deployment/` - Deployment

### Styling
- [src/css/custom.css](src/css/custom.css) - Global styles, Infima overrides, accessibility utilities, landing page animations
- [tailwind.config.js](tailwind.config.js) - TailwindCSS config with custom retro theme colors and animations
- TailwindCSS is integrated via a custom Docusaurus plugin in docusaurus.config.js
- `preflight: false` in Tailwind to avoid conflicts with Docusaurus/Infima CSS

### Theme Colors (defined in tailwind.config.js and custom.css)
```
--bg-primary: #0F1B2A (dark navy)
--color-cyan: #5CE1E6 (accent)
primary: #8f3aff (purple)
```

## Key Technical Decisions

- Vietnamese-only (`i18n.defaultLocale: "vi"`)
- Blog disabled in docusaurus.config.js
- React 19 with Docusaurus 3.9.2
- Mermaid diagrams enabled via @docusaurus/theme-mermaid
- Custom accessibility features: `.sr-only`, focus indicators, reduced motion support
