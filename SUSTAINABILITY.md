# Sustainability Plan

## Overview

Vietnamese Aiken is a community-driven open-source educational platform. This plan ensures the project continues to grow and serve Vietnamese developers learning Cardano smart contract development with Aiken.

## Governance

### Maintainer Structure

- **Core Maintainers** review all pull requests and make architectural decisions
- **Community Maintainers** are invited based on consistent, quality contributions
- See [MAINTAINERS.md](MAINTAINERS.md) for the current list

### Review Process

- Pull requests are reviewed within 3-5 business days
- New lessons require review from at least one core maintainer
- Smart contract changes must pass all tests (`aiken check`)

## Community Contributions

Contributors can participate in the following ways:

### Content
- Add new lessons covering advanced Aiken topics
- Improve existing tutorials with better explanations or examples
- Translate supplementary materials
- Submit corrections for Vietnamese language accuracy

### Code
- Add new smart contract examples with tests
- Improve the documentation site (Docusaurus components, styling)
- Fix bugs and improve accessibility
- Enhance CI/CD pipelines

### Review
- Submit feedback via [GitHub Issue templates](https://github.com/CardanoAikenVN/aiken-vn/issues/new/choose)
- Review pull requests from other contributors
- Test smart contract examples and report issues

## Funding

GitHub Sponsors is enabled to support continued development. Funds are used for:
- Hosting and infrastructure costs
- Workshop organization and promotion
- Video production (Vietnamese voiceover + English subtitles)
- Community events and outreach

See [.github/FUNDING.yml](.github/FUNDING.yml) for sponsorship configuration.

## Future Roadmap

### Short-term (3-6 months)
- Additional advanced lessons (multi-signature, governance contracts)
- More real-world smart contract examples
- Video tutorials for each curriculum part

### Medium-term (6-12 months)
- Community-led workshops (quarterly)
- Integration with Cardano testnet for hands-on exercises
- Guest lessons from Cardano ecosystem developers

### Long-term (12+ months)
- Certification program for course completers
- Expansion to cover other Cardano development tools
- Partnership with Vietnamese developer communities and universities

## Continuity

If core maintainers become unavailable:
1. Community maintainers assume review responsibilities
2. The CC BY-4.0 license allows anyone to fork and continue the project
3. All content and code remain publicly available on GitHub

## Metrics for Success

- Monthly active contributors (target: 5+)
- New lessons added per quarter (target: 2+)
- GitHub Issues resolved per month (target: 5+)
- Community channel active members (target: 50+)
- Workshop attendees per event (target: 30+)
